# Torii Kado Website v1.2

This is the complete static promotional website for Torii Kado.

## Fixes in v1.2
- Corrected the How It Works layout so descriptions no longer collapse into a narrow column.
- Increased the desktop content width and improved responsive grid sizing.
- Rebuilt the comparison slider using full 1350×1920 source and translated pages.
- The comparison now shows the complete page without cropping.
- Both sides remain the same size and alignment while the slider moves.
- The left side is the original Japanese page and the right side is the translated output.

## Open locally
Open `Torii-Kado-Landing-Page.html` for the self-contained version, or `index.html` for the standard folder-based website.

## Publish
Upload the complete folder to any static web host. See `PUBLISHING.md`.
