Important execution boundary
----------------------------
The supplied application delegates fully automatic OCR, inpainting and model translation to a separate manga-image-translator backend and a local Ollama model. Those Windows/local-model dependencies were not available in this Linux build environment. Therefore this report does not claim that the automatic OCR + Ollama stage ran here. The revised demo pages were cleaned, boxed and lettered through Torii Kado's actual project/editor rendering code and then displayed in the real app.
