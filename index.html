Complete manga workflow

Not just a translator.
A full workstation.

Import a volume, run OCR and local translation, inspect every page, repair difficult lettering, preserve story context and rebuild final ZIP/PDF exports.

01Import

Complete volume intake

Folders, images, PDF, ZIP/CBZ and RAR/CBR sources become resumable local projects.

02OCR

Multiple OCR pipelines

Neural OCR, Manga OCR, CTC, orientation passes, gamma, inversion and difficult-scan controls.

03Continuity

Story-aware translation

Preceding pages, project notes and fixed terminology guide names, pronouns, tone and relationships.

04Correction

Manga-first editor

Zoom, pan, erase, add text boxes, resize boundaries, rotate text, change fonts and hold the original.

05Self-check

Expandable page audit

Find untranslated text, overflow, tiny lettering, overlap and prompt leakage, then jump to the exact region.

06Selected area

Recover missed text

Draw around missed lettering, run dedicated local OCR, clean only that rectangle and create an editable translation.

07Hardware

GPU, low-VRAM or CPU

NVIDIA acceleration, system-RAM fallback and CPU-only operation support a wider range of Windows PCs.

08Delivery

Corrected ZIP & PDF

Edited pages take precedence, exports are marked stale after changes, and rebuilds use the latest page state.

09Languages

Multilingual interface

English, Spanish, French, German, Italian, Portuguese, Russian, Japanese, Korean and Simplified Chinese.

10Preservation

Raw result stays raw

The automatic backend PNG is retained byte-for-byte; corrections happen only when the user chooses them.

11Batch

Translate, retry or retranslate

Process remaining pages, retranslate selected pages, or rebuild all while archiving previous results.

12Models

Benchmark installed models

Compare local Ollama models on the actual PC before choosing the main translation model.
