FAQ

Know what to expect

Torii Kado provides a serious local translation and correction workflow. Results still depend on source quality and model choice.

Does Torii Kado upload my manga?

No cloud account is required. The application is designed around local OCR, image processing and compatible local LLM translation.

Can it fix text the OCR missed?

Yes. Draw around the missed lettering and use selected-area OCR to translate, clean and replace that region.

Will every page be perfect automatically?

No translation tool can guarantee that. Torii Kado provides story memory, validation, self-checks and detailed editing tools so you can identify and correct problems efficiently.

Can I retranslate only certain pages?

Yes. Use Translate Remaining, Retranslate Selected or Retranslate All depending on the project.

Which local model should I use?

Use Benchmark Installed Models in Settings, then choose the strongest model your hardware can run at a practical speed.

Which formats can I export?

Torii Kado can rebuild image archive outputs such as ZIP/CBZ and PDF workflows after translation and correction.
